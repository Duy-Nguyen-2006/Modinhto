Tôi muốn bạn tạo 1 file Python để crawl trang web xhamster https://vi.xhamster2.com
và tìm kiếm video theo tên diễn viên.

YÊU CẦU:
1. Chỉ tạo 1 file Python duy nhất
2. Sử dụng thư viện Crawl4AI (giống như trong file crawl4ai)
3. Sử dụng BeautifulSoup để parse HTML
4. Người dùng nhập tên diễn viên qua input()
5. Trả về danh sách video với: Tiêu đề và Link

CÁC BƯỚC THỰC HIỆN:

https://vi.xhamster2.com/pornstars/melody-marks

- Hàm main(): chạy chương trình
